Class: CsoundObj

CsoundObj()

This ES6 Class provides an interface to the Csound engine running on a node (an AudioWorkletNode where available, ScriptProcessorNode elsewhere) This class is designed to be compatible with the previous ScriptProcessorNode-based CsoundObj
