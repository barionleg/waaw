Web Assembly Audio Worklet Csound Examples

A collection of step-by-step examples using WAAW Csound, which also run on WASP (Web Assembly Script Processor) version of Csound, for browsers where AW is not yet available.

  • Random note generator: This example demonstrates how to load the WASM Csound library and use the CsoundObj class to compile and perform simple synthesis code that is embedded in the HTML page, connecting also to an AnalyserNode for waveform and spectrum display

  • Sliders: This example demonstrates how to use channels to send control data into Csound from HTML 5 sliders.

  • Minimal: Simple interactive example demonstrating csound.js with WAAW Csound.

  • Canvas: Interactive synth using 2-D canvas control, with csound.js.

  • MIDI: A simple MIDI example using HTML 5 buttons.

  • MIDI player: A GM soundfont MIDI file player.

  • CSD player: CSD file player.

  • Stria: Kevin Dahan's reconstruction of John Chowning's classic composition.
